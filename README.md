# Ultimate Glassmorphic Student Management System

A **futuristic PySide6 Student Management System** with a sleek **blackish glassmorphic UI**, perfect for your portfolio!  

### Features
- Rounded rectangular main window with elegant drop shadow  
- Blackish semi-transparent glass effect with blur aesthetics  
- Fully **draggable window** for smooth UX  
- Input fields auto-focus on Enter for faster data entry  
- Table displays students with first-column highlight  
- Buttons with subtle hover effects  
- JSON-based persistent storage for all student records  
- Ideal for **portfolio showcase** and GUI mastery
